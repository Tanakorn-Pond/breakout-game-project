# โปรเจกต์เกม 2D Breakout

เกม 2D แนว **Breakout**  
ผู้เล่นควบคุมแผ่นบอร์ดเพื่อเด้งลูกบอลไปทำลายบล็อก  
เป้าหมายคือทำลายบล็อกทั้งหมดโดยไม่ให้ลูกบอลตกออกจากฉาก

### เครื่องมือที่ใช้
- Visual Studio

### ภาษาที่ใช้
- C#

### ความสามารถของเกม
- ควบคุมแผ่นบอร์ดเพื่อเด้งลูกบอล
- ระบบคะแนน (Score Board)
- ระบบแพ้ / ชนะ
- ระบบหยุดเกม (Pause)
- โหมด Free Mode
- รีเซ็ตลูกบอล
- รีเซ็ตบล็อก
